Let's talk about basemaps v. feature layer maps

There are two primary spatial data types used in GIS:

  • Raster
  • Vector
  • Not always, but for the sake of simplicity, basemaps are usually rasters and feature layers are usually vectors.
  • We are concerned with manipulating and adding feature layers in this course.

Common Vector Data Types

  • Esri Shapefile
  • Geographic JavaScript Object Notation (GeoJSON)
  • Google Keyhole Markup Language (KML/KMZ)
  • OpenStreetMap OSM XML

A more comprehensive list if you're curious

Honorary mention: Comma Separated Values (.CSV)

Vectors are composed of:

  • Points
  • Lines
  • Polygons

Points:

  • Each point feature is represented as a single coordinate pair (x,y)
  • No area, no length
  • Visual Options: change size, color, symbol shape

Lines

  • Represented as a series of x,y points with 2 or more connected vertices
  • No area, but has length
  • Visual Options: line style, line thickness, or color

Check: What unit was length measured in? Is this the same unit the map is projected in?

PS: GeoJSONs are not tabular. GIS programs display their attribute tables tabularly for convenience, but we will explore GeoJSONs in depth later in the course.

Polygons

  • Enclosed shape with 3 or more vertices
  • Has area
  • Visual Options: fill color, fill pattern, outline color, outline pattern

Attribute types

  • Integer:
    • non-fractional numbers
    • 0,1,2,8009, etc
  • Real:
    • fractional numbers
    • 0.13, 2.5, 3.0, etc
  • String:
    • a “string” of characters
    • “My dog is barking”, “0134”, “I have 17 dogs”

Note that numerical values can be stored as strings
