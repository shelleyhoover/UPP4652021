Course Overview

Course Goals

  1. By the end of this course students will be able to use ArcGIS Online & Leaflet.js to create interactive, online maps
  2. Students will have a deeper knowledge of effective map design & functionality
  3. Equip students with basic working knowledge of HTML, CSS, and JavaScript
  4. Provide students with an overview of the current Web GIS landscape

This course is focused on visualization.

We will not focus on data management or statistics, however you should be conscientious of how these affect your final product

Data will be provided for the course, but you are highly encouraged to use your own!

If you need help cleaning/geocoding data from other courses or projects, reach out to me.

    How to Lie with Maps

    First, a quick refresher (or crash course) on Thematic Maps

    We will create interative thematic maps in this course, as opposed to general reference or topographical maps.

    Thematic maps are maps designed to show a particular theme

    Basic types of thematic maps

    • Choropleth
    • Proportional Symbols
    • Cartograms
    • Dot
    • Flow Map

    Choropleth Maps use color or shading to show data over a defined area.

    You'll see a lot of these.

    Hull House example

    Proportional Symbols Maps use symbols of varying sizes to show a quantitative value

    Another proportional symbols example

    Cartograms distort land area to represent a variable.

    Color used to represent second variable (and third variable!)

    Dot Density Maps use a dot to represent the distribution (or density) of a phenomenon

    Color used to represent second variable

    The famous John Snow Cholera map

    Flow maps show the movement of a unit or variable.

    Color & symbol size used to represent second & third variables

    Color used to represent second variable

    Minard's flow map of Napoleon's 1812 Russian campaign.

    Symbol size used to represent second variable.

    Everyone making a map is doing so for a reason.

    Some maps are a bit less subtle about their agendas than others

    Spooky!

    I was not aware this was a concern

    all maps 'lie'

    It is impossible* to have a completely objective map

    Even if your goal is to create an objective data visualization, every map maker needs to make choices that effectively result in the map telling lies.

    *Okay, a map with a 1:1 scale could be objective. But it would be pretty useles

    Despite the obvious agenda of one, both can be considered honest depictions the 2016 election results.

    You're free to make map design choices that support your message. But if you're going to, do it on purpose.

    If your goal is to make as objective a map as possible, consider why you're making certain design choices.

    Either way, make your manipulative design a bit more subtle than this (and also don't literally lie...)

    Until you're out in the world pushing agendas, let's be conscientious of the following map design choices that can be deceptive:

  • How data is areally aggregated
  • How data is classed
  • Assuming the viewer has the literally the same worldview as you
  • Color choices
